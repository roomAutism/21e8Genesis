# Autism Room Inc.

Minimal static site for Autism Room Inc.

## Deploy to Vercel

1. Upload this directory to a Git repository, or import the folder directly in Vercel.
2. In Vercel, choose **Other** as the framework preset.
3. Leave the build command and output directory empty, then deploy.

Vercel serves `index.html` directly. No build step or environment variables are needed.
